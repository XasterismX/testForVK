# Мониторинг простого веб-приложения

## Описание
Представлена простая система мониторинга Go веб-приложения, которая:
- Запускает веб-сервер на порту 8080, отвечающий "Hello World!"
- Мониторит доступность приложения через curl
- В случае недоступности автоматически перезапускает приложение
- Мониторинг реализован в bash-скрипте с логированием работы
- Запуск мониторинга автоматизирован через systemd timer с периодом проверки

## Как использовать
1. Отредактируйте пути в `monitor.sh` и `install.sh` под ваш каталог
2. Запустите `sudo install.sh` для установки и настройки автозапуска
3. Логи мониторинга находятся в `/var/log/test_server_monitoring.log`

## Параметры
- Порт приложения: 8080
- Период проверки в таймере systemd: 30 секунд (можно изменить в файле app_monitor.timer)

## Требования
- curl, bash, systemd
- Права root для установки сервисов
